# Trafico

Liga A. - это адаптивный лендинг с использованием gulp, html (+ БЭМ нейминг), pug, scss и js (ES6)

Для запуска:

1. Клонируем репозиторий
 ```
 git clone https://github.com/cool-hiya/trafico.git
 ```
2. Устанавливаем node_modules
 ```
 npm install
 ```
3. Запускаем
 ```
 npm run start
 ```
